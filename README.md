# Particle-Based Modeling - Траектория движения частиц в воздушном потоке

## Описание

Проект представляет собой симуляцию движения аэрозольных частиц в неоднородном воздушном потоке. Используется численное моделирование методом Рунге-Кутта 4-го порядка для расчёта траекторий частиц с учётом сил аэродинамического сопротивления. Программа вычисляет плотность распределения частиц в пространстве и сохраняет результаты для визуализации.

## Физика симуляции

Программа вычисляет траектории частиц на основе следующих параметров:

### Основные силы:
- **Сила сопротивления (Drag Force)**: $F_d = \frac{1}{2} \pi r^2 C_d \rho_{air} |v_{rel}|^2$
  - где $r$ - радиус частицы
  - $C_d$ - коэффициент лобового сопротивления (зависит от числа Рейнольдса)
  - $\rho_{air}$ - плотность воздуха
  - $v_{rel}$ - относительная скорость частицы относительно воздуха

### Число Рейнольдса:
$$Re = \frac{2 \rho_{air} |v_{rel}| r}{\mu}$$

### Коэффициент сопротивления:
$$C_d = \left(0.325 + \sqrt{0.124 + \frac{24}{Re}}\right)^2$$

## Параметры симуляции

### Свойства воздуха:
- Плотность: 1.225 кг/м³
- Температура: 288 K (15°C)
- Давление: 101325 кПа
- Вязкость: 1.75×10⁻⁵ Па·с
- Скорость ветра: можно выбрать поле

### Свойства частиц:
- Плотность материала: 2500.3 кг/м³
- Минимальный радиус: 50 мкм
- Начальная позиция: случайная в диапазоне (-10, 10)
- Начальная скорость: 0 м/с
- Случайный радиус: 50 мкм + случайная добавка

### Поля скорости ветра:
Проект поддерживает 4 различных поля скорости ветра (задается U_FUNCTION):
1. **Константное поле**: U = (10, 5) м/с
2. **Переменное поле #1**: зависящее от sin и exp функций
3. **Переменное поле #2**: с cos и sin компонентами
4. **Переменное поле #3**: вихревое поле

## Компиляция и запуск

### Требования:
- C++11 или выше
- g++ компилятор
- Boost библиотеки (boost/numeric/odeint)
- Python 3.6+ (для визуализации)

### Установка зависимостей:

**Boost библиотеки (Ubuntu/Debian):**
```bash
sudo apt-get install libboost-all-dev
```

**Python зависимости:**
```bash
pip install numpy matplotlib python-dotenv
```

### Компиляция C++ программы:
```bash
g++ -std=c++17 main.cpp -o main
```

Или с указанием количества частиц:
```bash
g++ -std=c++17 -DNUM_PARTICLES=1000 main.cpp -o main
```

### Запуск симуляции:
```bash
./main
```

### Создание видео из результатов плотности:
```bash
python density_video.py
```

### Визуализация траекторий (для отладки):
```bash
python graphics.py
```

## Конфигурация

Настройки задаются в файле `.env`:

```env
PARTICLES_NUM=200000    # Количество частиц
U_FUNCTION=4            # Номер поля скорости ветра (1-4)
```

### Параметры симуляции:
- **Сетка**: 60×60 ячеек
- **Область**: от -30 до +30 по X и Y
- **Временных шагов**: 5000
- **Шаг интегрирования**: 1e-3
- **Метод интегрирования**: Runge-Kutta 4-го порядка

## Выходные данные

Программа генерирует файлы плотности в директории `density/`:
- `density_0.txt` - плотность частиц на временном шаге 0
- `density_1.txt` - плотность частиц на временном шаге 1
- и т.д.

Каждый файл содержит матрицу 60×60 с количеством частиц в каждой ячейке сетки.

### Дополнительные выходы:
- **Максимальная плотность**: выводится в консоль для настройки визуализации
- **Время выполнения**: отображается прогресс и общее время симуляции
- **Видео**: создается с помощью `density_video.py`
