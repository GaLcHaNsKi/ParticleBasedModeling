# Particle-Based Modeling - Траектория движения частиц в воздушном потоке

## Описание

Проект представляет собой симуляцию движения аэрозольных частиц в неоднородном воздушном потоке. Используется численное моделирование для расчёта траекторий частиц с учётом сил аэродинамического сопротивления.

## Физика симуляции

Программа вычисляет траектории частиц на основе следующих параметров:

### Основные силы:
- **Сила сопротивления (Drag Force)**: $F_d = \frac{1}{2} \pi r^2 C_d \rho_{air} |v_{rel}|^2$
  - где $r$ - радиус частицы
  - $C_d$ - коэффициент лобового сопротивления (зависит от числа Рейнольдса)
  - $\rho_{air}$ - плотность воздуха
  - $v_{rel}$ - относительная скорость частицы относительно воздуха

### Число Рейнольдса:
$$Re = \frac{2 \rho_{air} |v_{rel}| r}{\mu}$$

### Коэффициент сопротивления:
$$C_d = \left(0.325 + \sqrt{0.124 + \frac{24}{Re}}\right)^2$$

## Параметры симуляции

### Свойства воздуха:
- Плотность: 1.225 кг/м³
- Температура: 288 K (15°C)
- Давление: 101325 Па
- Вязкость: 1.75×10⁻⁵ Па·с
- Скорость ветра: 10 м/с (x), 5 м/с (y)

### Свойства частиц:
- Плотность материала: 2500.3 кг/м³
- Минимальный радиус: 50 мкм
- Начальная позиция: (0, 0)
- Начальная скорость: случайная, от -20 до 20 м/с

## Структура проекта

```
ParticleBasedModeling/
```

## Компиляция и запуск

### Требования:
- C++11 или выше
- g++ компилятор
- Python 3.6+ (для визуализации)

### Установка зависимостей Python:
```bash
pip install numpy matplotlib python-dotenv
```

### Компиляция C++ программы:
```bash
g++ main.cpp -o main
```

### Запуск симуляции:
```bash
./main
```

или с указанием количества частиц:
```bash
NUM_PARTICLES=500 ./main
```

### Визуализация результатов:
```bash
python graphics.py
```

или с указанием количества частиц:
```bash
PARTICLES_NUM=100 python graphics.py
```

## Конфигурация

Количество частиц задаётся в файле `.env`:

```env
PARTICLES_NUM=100
```

## Выходные данные

Программа генерирует файлы траекторий в директории `trajectories/`:
- `trajectory_0.txt` - траектория частицы 0
- `trajectory_1.txt` - траектория частицы 1
- и т.д.

Каждый файл содержит два столбца: координаты X и Y в каждый момент времени.

## Пример использования

```bash
# Скомпилировать
g++ -std=c++17 main.cpp -o main

# Запустить симуляцию с 100 частицами
./main

# Визуализировать результаты
python graphics.py
```
